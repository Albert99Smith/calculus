\begin{enumerate}[label=(\alph*)]
\item If $\vec{u}$ is a nonzero vector, then $\frac{\vec{u}}{\left|\vec{u}\right|}$ is a unit vector.
\item If $\vec{u} = \vec{v}$ then $\left|\vec{u}\right| = \left|\vec{v}\right|$.
