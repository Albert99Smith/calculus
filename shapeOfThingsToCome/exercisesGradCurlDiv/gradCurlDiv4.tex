In the exercises that follow, consider the function $\vec{F} : \R^3 \to \R^3$ defined by the rule $F(x,y,z) = \vector{x,y^2,z^3}$.

\begin{exercise}
  Compute \(\divergence F\).
  \begin{multipleChoice}
    \choice{This is zero.}
    \choice[correct]{This can be computed (and is nonzero).}
    \choice{That does not even make sense.}
  \end{multipleChoice}
  \begin{exercise}
    \[
      \divergence F = \answer{ 3 z^2 + 2  y + 1 }
    \]
  \end{exercise}
\end{exercise}
